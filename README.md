Currency Converter

A simple and user-friendly Currency Converter web application that allows users to convert amounts between different currencies using real-time exchange rates. The application provides accurate conversions with a clean and responsive interface, making it easy to check currency values instantly.

Features
Convert between multiple international currencies.
Real-time exchange rate updates.
Simple and responsive user interface.
Fast and accurate currency calculations.

Technologies Used
HTML
CSS
JavaScript
Exchange Rate API

How to Use
Enter the amount to convert.
Select the source currency.
Select the target currency.
Click on Get Exchange Rate to view the converted amount instantly.

This project was built to practice API integration and front-end web development concepts while creating a useful real-world application.
